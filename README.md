# mm

## Project Structure

- `index.html`, `research.html`, `publications.html`, `talks.html`, `resume.html`, `contact.html`: site pages
- `assets/images/`: all images and media assets
- `assets/files/`: CV and downloadable files
- `assets/js/`: JavaScript files
- `_layouts/`, `_sass/`: Jekyll layout and style sources
- `_site/`: generated site output
